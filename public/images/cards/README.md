# Instructions for Cards

##############################

### Cards

1. Go to https://www.pokecardgenerator.com/
2. Select Scarlet/Violet
3. Template: Trainer Full Art
4. Trainer Type & Trainer Type Labels are dependant on the type of card being made:

- Education is Stadium
- Project is Tool
- Experience is Supporter
- Hobby is Item

5. If the Trainer Type has a Rule, make sure to update the copy to replace the type with the new type
6. Illustrator is Joshua Bee
7. Export the largest size possible
8. Resize to the sizes we want
9. Convert to WebP format
10. Also add the image used on the card to the folder as well
